Methods¶

This module automates the separation of body part and reference part of a LateX produced PDF.

separate_body_ref._clean(dest, level, recursive, rmFileExtensions, rm)[source]¶

Remove certain files in a specific directory

This function removes the files whose extensions are in the list rmFileExtensions by using rm method. More specifically, this function will recursively remove certain files (i.e., files with extensions contained in rmFileExtensions)

Parameters:

dest : str

The directory

level : int

Current level

recursive : int

Maximum recursive depth

rmFileExtensions : set

A set of string specifies the extensions of files to be deleted

rm : object

An object that is callable and accept a single string parameter

texFileLists : list

The tex file list

Dependency¶

Lists of dependent packages:

  1. Send2Trash

Usage¶

Please this script in the same directory as the main tex file for your latex project. And then run this script by typing python ./separate_body_ref.py -h you will get the details of how to use this script.

The tex example is copied from Creating separate pdf file containing the bibliography with minor revisions.
